# 📱 iPhone Sales Data Analysis Using Python

## 📖 Project Overview

This project focuses on analyzing iPhone sales data using Python. The analysis helps identify sales performance, revenue trends, top-selling models, country-wise sales distribution, and customer payment preferences through data analysis and visualization techniques.

---

## 🎯 Objective

The main objective of this project is to extract meaningful business insights from iPhone sales data and visualize patterns that support data-driven decision-making.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset Features

The dataset contains information such as:

- Sale Date
- iPhone Model
- Country
- Quantity Sold
- Price
- Payment Method

---

## 📊 Analysis Performed

### Data Cleaning
- Handled missing values
- Converted data types
- Processed date columns

### Feature Engineering
- Created Revenue column
- Extracted Month from Sale Date

### Exploratory Data Analysis (EDA)
- Revenue Analysis
- Country-wise Sales Analysis
- Model-wise Sales Analysis
- Payment Method Analysis
- Monthly Revenue Trend Analysis

---

## 📈 Visualizations

- Country-wise Revenue Bar Chart
- Model-wise Revenue Bar Chart
- Payment Method Pie Chart
- Monthly Revenue Trend Line Chart

---

## 💡 Key Insights

- Identified top-performing countries based on revenue.
- Found the best-selling iPhone models.
- Analyzed customer payment preferences.
- Evaluated monthly revenue trends.
- Generated business insights from sales performance.
